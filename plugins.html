Plugins

Plugins are simply bits of Lisp code with a few requirements for file location and name. By default plugins are located in one of two locations: 

    
     1. cyco/plugins/                  The default plugins shipped with CYCO.
     2.  ~/.config/cyco/plugins/        Personal plugins.

A plugin named 'foo' must meet the following conditions.

  1. A directory with the plugin's name must exists in either of the locations above.
  2. The directory and names of all files it contains must be lowercase only.
  3. The main file for a plugin 'foo' is foo-main.lisp
    
    ~/.config/cyco/plugins/foo/
                            |
                            +-- foo-main.lisp
                            +-- additional lisp files
                            +--
(PLUGIN name &optional reload)

Loads plugin.

    name should be an unquoted symbol.
    	
    Once a plugin has been loaded it is not reloaded unless the optional
    reload flag is true.
    	
    A plugin may load another plugin.
(LOAD-PLUGIN-FILE name)

Loads lisp file relative to current plugin directory.

(FORGET-PLUGINS)

Forgets all currently loaded plugins.

    The plugins are not actually unloaded from memory. Calling FORGET-PLUGINS 
    simply removes the record of all plugins having been loaded.
(?PLUGINS)

Prints list of all plugin names.

    An asterisks indicates which plugins have been loaded.
