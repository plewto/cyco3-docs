String Utilities

(CLONE string &key &allow-other-keys)

Returns string

(FINAL string)

Returns final character of string

    (FINAL “abc”) → “c”
(BUTFINAL string)

Returns string with final character removed.

    (BUTFINAL “abc”) → “ab”
(CNTH n string)

Returns nth character of string using cyclical indexing.

    
    (CNTH 2 “abcd”) → “c”
    (CNTH 6 “abcd”) → “c”
(STR+ &rest args)

Concatenate string representation of args.

    
    (STR+ “ABC” 'zac) → “ABCZAC”
(SPACES (&optional (n 1)))

Returns string consisting of n spaces.

(SCOPIES (n &optional (char #\-)))

Returns string consisting of n copies of character char.

(PARSE-WORD string &optional (delim #\space))

Separates first delimited word from string.

    
    Returns two values:
       1.  The delineated word
       2.  The remaining string 
    The remaining value is nil when argument contains a single word.
(SPLIT-STRING string &optional (delim #\space))

Returns list of delineated word in string.

(STARTS-WITH-P string target)

Returns true if initial characters of string are string= to target.

(SFORMAT frmt &rest args)

Convenience version of FORMAT.

    (SFORMAT foo...) is equivalent to (FORMAT nil foo...)
(FORMAT-BINARY n &key (bits 16) group)

Converts number to binary string

    
    (BINARY-FORMAT 14)   → “0000000000001110”
    (BINARY-FORMAT 14 :GROUP 4) → “0000 0000 0000 1110”
(PALINDROME string &key (elide nil))
    (PALINDROME “ABC”) → “ABCCBA”
(PICK string)

Returns random character from string.

(CENTER-STRING test width &optional (shift 0))

Centers text in new string with length width.

    
    text - The text
    width - length of result.
    shift - optional amount to shift text right.
    
    (CENTER-STRING “ABC” 9) → “   ABC   ”
