A CUELIST is a list of event time specifications. The exact format is not specified but the BAR format of (bar beat subbeat) is a de facto standard. See Cuestrings.

A CUESTRING is an alternate means of specifying a cuelist using a string. The exact format is not specified but a binary form used by BINCUE is a de facto standard.

A MIDI EVENT is a cons of form (time . message) where time is a real number in seconds and message is a MIDI message

A TIME SIGNATURE DESIGNATOR may be one of the following

  • An instance of TIME-SIGNATURE
  • A list of form (bar-count beat-count)
  • NIL - use default, will be one of
    1. The current-section of *project*.
    2. If there is no current-section, use *project*.
    3. If *project* is nil, use a fallback signature, 2-bars in 4/4 time.
