Meta Text Events

Meta-text events are inserted into a section by the TEXT-PART macro.

(MAKE-TEXT-PART name &key ...)
(TEXT-PART name &key ...)
    MAKE-TEXT-PART and TEXT-PART are identical except TEXT-PART
    binds the new instance to the symbol name while MAKE-TEXT-PART does not.
    The name argument should be a quoted symbol for make-text-part and
    unquoted for text-part.
    
         name     - symbol
         :section - parent section, defaults to current section.
         :cuefn   - cueing function, defaults to section value
         :shift   - time shift, defaults to 0 seconds.
         :remarks - optional remarks text
         :events  - text-event specifications, see below.
    
    The events is a nested list of form:
    
         :events '((time-1) type text...)
                   (time-2) type text...)
                    ....................
                   (time-n) type text...))
    
    Where time is in a format expected by the cue function.  The default time
    specification is a list (bar beat sub-beat).
    
    type is a keyword indicating the type of text event.   Possible values are
    
          :copyright
          :cue
          :instrument-name
          :lyric
          :marker
          :text
          :track-name
