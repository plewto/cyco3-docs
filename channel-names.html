(SET-META-CHANNEL name channel &optional remarks)

Makes symbolic assignment to a MIDI channel.

        A meta-channel may be defined in terms of another channel 
        so long as they ultimately resolve to a 'real' channel number.
    	
    	name    - symbol
    	channel - integer 1 <= channel <= 16
    	          symbol, an indirect channel assignment.
    	remarks - optional remarks text.
    	
    	Examples:
    	
    	(SET-META-CHANNEL 'oberheim 1)
    	(SET-META-CHANNEL 'drum-machine 10)
    	(SET-META-CHANNEL 'snare 'drum-machine)  ;; snare defined in terms of drum machine.
(META-CHANNEL channel &optional (resolve nil))

Returns value of meta-channel.

        By default returns one-level of
        indirection.  If resolve is true, returns actual MIDI channel.  
    	
    	Using examples above:
    	
    	(META-CHANNEL 'snare)   → drum-machine
    	(META-CHANNEL 'snare t) → 10
        (META-CHANNEL 3)        → 3
(META-CHANNEL-NAMES channel)

Returns list of names for channel.

(META-CHANNEL-ASSIGNMENT-P object)

Predicate, true if object is a channel assignment.

(?CHANNELS)

Displays list of channel assignments.


MIDI Controller Names

(DEFINE-CONTROLLER name controller-number)

Assign name to MIDI controller number.

(DEFINED-CONTROLLERS)

Returns list of assigned controller names.

(?CONTROLLERS)

Displays assigned controller names.

(GET-CONTROLLER-NUMBER name &key (default nil))

Returns controller number assigned to name.

    If name is not an assigned controller, return default.
Default Controller Names
        BANK-SELECT   0
        WHEEL         1
        BREATH        2
        FOOT          4
        PORT-TIME     5
        VOLUME        7
        PAN          10
