More Generators

Hailstone

(HAILSTONE seed &key even odd hook)

Returns generator which produces a hailstone sequence A061641

    
    seed  - Integer, starting value.
    :even - Function applied to even result, default (lambda (n)(/ 2 n))
    :odd  - Function applied to odd result, default (lambda (n)(1+ (* 3 n)))
    :hook - Function applied to output value, default (lambda (n) n)

Logistic

Image credit

(LOGISTIC &key prerun seed mu hook)

Returns generator which uses a Logistic map.

    
    :prerun - Integer, number of time to step generator after it is created or reset, default 0.
    :seed   - Float, initial value, 0.0 < seed ≤ 1.0, default 0.5
    :mu     - Float, 'fecundity' parameter, 0.0 ≤ mu ≤ 4.0, default 3.25
    :hook   - Value hook function, default (lambda (n) n)
