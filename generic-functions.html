(->cycle object)

Coerce object to Cycle pattern.

  • If object is an instance of Cycle, return it.
  • If object is some non-cycle Pattern, return a new Cycle with the elements of object.
  • If object is a list or vector, return new Cycle :of object.
  • For all other types, coerce object to a list and return new Cycle.

(->list object)

Coerce object to a list.

  • If object is a list, return it.
  • If object is some other sequence type, return new list of object's elements.
  • For all other types, create new list with object as its only element.

(->midi contents &key filename offset repeat pad no-stripe)

Creates new MIDI file from contents.

    
        :filename - Optional String, if not specified the filename is derived from
                    the name of contents and the current project.
        :offset   - float, initial offset time in seconds, default 0.
        :repeat   - int, number of times to repeat contents, default 1.  If repeat is too 
                    large stack depth may be exhausted.           
        :pad      - float, number of seconds to add to end of file, default 2.
        :no-stripe - DEPRECIATED, has no effect.
(->pattern object &key pattern-type)

Coerce object to Pattern of type pattern-type.

  • If object is a pattern-type, return it.
  • If object is a non pattern-type Pattern, return new instance of pattern-type with elements taken
  • from object.
  • For all other types return (pattern-type :of object)
  • The default pattern-type is Cycle.

(->string object)

Returns string representation of object. ->string is defined for following types

  • TIME-SIGNATURE
  • SMF
  • MIDI-KEY-SIGNATURE
  • MIDI-TIME-SIGNATURE
  • MIDI-TEMPO-MESSAGE
  • MIDI-META-TEXT
  • MIDI-META-MESSAGE
  • MIDI-SYSTEM-EXCLUSIVE
  • MIDI-SYSTEM-COMMON-MESSAGE
  • MIDI-CHANNEL-MESSAGE
  • MIDI-MESSAGE
  • CYCO-NODE
  • PATTERN
  • LIST
  • STRING returns string
  • T catchall

(->symbol arg &optional package)

Interns new symbol using name of object

(->vector object

Coerce object to vector.

  • If object is a vector, return it.
  • If object is a non-vector sequence, return new vector using elements of object.
  • For all other cases return new vector with object as its only element.

(clone mother &key &allow-other-keys)

Returns a deep copy of mother.

    Clone is defined for a wide array of types.  Where it is not defined,
    it returns the mother argument.  Depending on the type different sets of
    keyword arguments may be available.
    
    :new-name   - For named objects, format string applied to translate mother's name to daughter.
    :new-parent - For node objects, assign a new parent to the clone. If not
                  specified the parent of mother is used.
    
    For SECTION objects
    
    :exclude       - List of part names to be excluded.
    :rename-parts  - If non-nil all parts in cloned object will be renamed.
    :bind          - If true all cloned parts will be bound to symbols eq to their name.
    
    Cloned sections become the project's current-section.

(cuelist object &key form timesig use-subbeats)

Returns object's cuelist or changes format of a cuelist.

    
    object   - PART, cuelist or cuestring
    
    :form    - Result format, may be either :binary for a cuestring or :list (default) for a cuelist.
    :timesig - Time signature designator
    :use-subbeats - Sets reference time unit either to time-signature's subbeats or tsubbeats. Default t

(duck cuelist object &key invert timesig use-subbeats)

Creates a new cuelist by ducking objects events.

    
    cuelist - The source cuelist. 
    object  - Object to be ducked.
    :invert   - Boolean, if true return the AND of the two cuelist. Default nil
    :timesig  - Time signature designator.
    :use-subbeats - Default t.

    If child is a symbol, returns the node, if it exists, with a matching name.
