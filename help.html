Help

CYCO includes several help functions, these all begin with a question mark.

(??)

Display list of help functions. The list is not comprehensive.

(? object)

Display information about object, The format is dependent on the type of object.

(?a symbol &optional (package :cyco))

Alias for the Common Lisp apropos function.

Do not quote the argument. Use (?a foo) not (?a 'foo)

(?d symbol)

Alias for the Common Lisp describe function.

Do not quote the argument. Use (?d foo) not (?d 'foo)

(?o &optional (root +root-instrument+))

Display structure of the orchestra.

(?p)

Display structure of the current project.

(?chords &optional (chord-model *chord-table*))

List available chord types.

(?channels)

List assigned MIDI channel names.

(?controllers)

List assigned MIDI controller names.

(?kmap instrument)

Print details about the instrument's keynumber-map.

(?pmap instrument)

Print details about the instrument's program-map.

(?plugins)

Print list of available plugins.

(?projects)

Print list of all projects found in the top-level cyco-projects directory.
