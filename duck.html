Cuelist Ducking

Ducking uses one cuelist or part to modify another cuelist.

(DUCK cuelist mask &key invert &allow-other-keys)

Creates a new cuelist consisting of the elements of the argument cuelist which are not elements of the mask. If invert is true the result is the elements common to both cuelist and mask.

  • :INVERT NIL (the default) for cuelist C and mask M, selected entities are C AND (NOT M).
  • :INVERT T, selected entities are C AND M.

The cuelist argument must be in the BAR format ((BAR BEAT SUBBEAT ) ...)

The mask argument may be one of:

  • BAR format cuelist
  • String in binary format “110011...”, whitespace is ignored.
  • A PART
    
    (param hat-cuelist '((1 1 1)(1 1 3)(1 2 1)(1 2 3)(1 3 1)(1 3 3)(1 4 1)(1 4 3)))  ;; eighth notes
    (param kick-cuelist '((1 1 1)(1 2 1)(1 3 1)(1 4 1)))                             ;; down beats
    
    (duck hat-cuelist kick-cuelist) → ((1 1 3)(1 2 3)(1 3 3)(1 4 3))                 ;; up beats only
    (duck hat-cuelist kick-cuelist :invert t) → ((1 1 1)(1 2 1)(1 3 1)(1 4 1))
