Math Utilities

(2+ n)
(3+ n)
(4+ n)

Adds 2, 3 or 4 respectively to argument.

(LIMIT n mn mx)

Limits value to numeric range.

    Returns value n such that:   mn ≤ n ≤ mx
(APPROXIMATE value &key (scale 0.1) (min -1e9) (max 1e9) (epsilon nil))

Returns a number approximately equal to value.

    
        value - the original value.
        :scale - maximum deviation from value.
        :min - minimum result.
        :max - maximum result.
        :epsilon - see below.
    
    Using the default scale of 0.1, the result will be within 10 percent of value.
    
       0.9 * value ≤ result ≤ 1.1 * value
    
    When the original value is 0 the result is ±epsilon.  By default
    epsilon is set to the value of scale.
(MEAN list)

Returns artihmitic average of numeric list.

    (MEAN '(10 20 30)) → 20.
(DIVIDESP A B)

Predicate, returns true if b evenly divides a, where b ≠ 0.

    (DIVIDESP 12 3) → T
    (DIVIDESP 12 5) → NIL
(PARITY n &key (bits 64))

Returns even/odd parity for binary value n

    (PARITY #B1000) → 1
    (PARITY #b1100) → 0
(N-COMPLEMENT (gamut list))

Returns list of all number in gamut which are not elements of list.

    (N-COMPLEMENT '(2 3 5 7)(1 3 5 7 9 )) → (2)
