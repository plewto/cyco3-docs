Alloy

An ALLOY is a generator which combines the values of two embedded generators using a binary-function. The default function is addition.

(ALLOY a b &key function hook monitor action)

Returns new ALLOY object.

    
    a - Generator, first operand to the binary-function
    b - Generator, seconds operand to the function.
    :function - Function, binary function, default #'+
    :hook     - Function applied to the internal-value.
                Calling (internal-value alloy) returns (function a b)
                Calling (value alloy) returns the composition (hook (function a b))
                Default (lambda (n) n)
    :monitor  - Predicate function called within NEXT-1. The action function 
                is executed whenever monitor is non-nil.
                Default (lambda (value) nil)
    :action   - Function called within NEXT-1 whenever monitor is non-nil.
                Default (lambda (alloy value) value)
                The result becomes the next internal-value of the alloy.
    
    Examples:
    	     
    (param a (counter 0 100 :by 10))
    (param b (cycle :of '(0 1 2)))
    (param c (alloy a b))
    (next c 10) → (0 11 22 30 41 52 60 71 82 90)
