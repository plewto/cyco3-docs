System Exclusive Events

The SYSEX inserts System Exclusive messages into a Section.

(MAKE-SYSEX-PART name &key ...)
(SYSEX-PART name &key ...)
    MAKE-SYSEX-PART and SYSEX are identical except SYSEX binds the new part to
     the symbol name while MAKE-SYSEX-PART does not.  name should be a quoted
     symbol for MAKE-SYSEX-PART and unquoted for SYSEX.
    
        name     - symbol
        :section - parent section, defaults to current section.
        :cuefn   - cue function, defaults to section value.
        :shift   - time shift amount, defaults to 0 seconds.
        :render-once - Boolean, if non-nil do not repeat events while rendering.
        :remarks - optional remarks text
        :events  - list of event specifications, see below
    
    events are a nested list of form 
    
        ((time-1) data...)
         (time-2) data...)
          ...............
         (time-n) data...))
    
    Where time is in a format accepted by the cue function,  the default format
    is (bar beat sub-beat)
    
    data is System Exclusive data bytes.   Do not include the SYSEX or
    END-SYSEX status bytes.   Each data is a 7-bit byte   0 ≤ data ≤ 127.
