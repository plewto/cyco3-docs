Euclidean Rhythms

Euclid has been reimplemented as a set of functions. Originally it had been implemented as a Generator. The original Euclid generator has been moved to a plugin called 'LEGACY-EUCLID'.

See Euclidean Rhythm Wikipedia page. For an online demo see here.


(EUCLID->BINARY-CUELIST length points &key shift invert)

Creates binary cuelist with a Euclidean rhythm.

    
    length  - Integer, number of possible events.  0 < length.
    points  - Number of events, may be an integer, float or nil.
              * Integer, absolute number of events.  0 < points ≤ length.
              * Float, number of events relative to length.  0.0 < points ≤ 1.0
              * NIL, random number of events.
    :shift  - Amount of pattern rotation, may be an integer, float or nil.
              * Integer, absolute. Default 0
              * Float, shift relative to length.  0.0 < points ≤ 1.0
              * NIL, random shift.
    :invert - Boolean, if true invert event selection.
    
    (EUCLID->BINARY-CUELIST 8 5)            → “10110110”
    (EUCLID->BINARY-CUELIST 8 5 :INVERT T ) → “01001001”
    (EUCLID->BINARY-CUELIST 8 5 :SHIFT 4)   → “01101011”
(EUCLID points &key shift invert timesig use-subbeats)

Creates Euclidean cuelist in bar format: ((BAR BEAT SUBBEAT)...)
The rhythmic length is set by the timesig argument.

    
    points   - Number of events. May be integer, float or nil as per EUCLID->BINARY-CUELIST function.
    :shift   - Rhythmic shift. May be integer, float or nil as per EUCLID->BINARY-CUELIST function.
    :invert  - Boolean, if true invert selected events.
    :timesig - Reference time-signature. May be one of the following.
               * An instance of TIME-SIGNATURE
               * A list of form (bars beats subbeats)
               * NIL (the default).
    
               The default is to first try the current-section of the global *PROJECT*.
               If there is no current-section, then *PROJECT* is used.
               If *PROJECT* is nil, a fallback 2-bar time-signature in 4/4 is used.
    
    :use-subbeats - Boolean.  If true use the time-signatures subbeats as the reference time-unit.
                    Otherwise use tsubbeats.  Default true.
    
    
    (euclid 5 :timesig '(1 4 4)) → ((1 1 1) (1 1 4) (1 2 3) (1 3 3) (1 4 2))
(EUCLID2 p1 p2 &key s1 s2 inv1 inv2 split timesig use-subbeats)

Splices two Euclidean cuelist together.

    
    p1     - Rhythm 1 points. May be integer, float or nil as with EUCLID.
    p2     - Rhythm 2 points.
    :s1    - Rhythm 1 shift. May be integer, float or nil as with EUCLID.
    :s2    - Rhythm 2 shift.
    :inv1  - Rhythm 1 invert flag. 
    :inv2  - Rhythm 2 invert flag.
    :split - Splice point bar number. Defaults to half time-signature bar-count.
    :timesig - Reference time-signature as with EUCLID function.
    :use-subbeats - Boolean, as with EUCLID function.
